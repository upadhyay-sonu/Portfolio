### 1. Install
```bash

```

### 2. Run
```bash
npm run dev
```

### 3. Open
Visit `http://localhost:3000`

## Build & Deploy

### Build for Production
```bash
npm run build
npm start
```

### Deploy to Vercel
```bash
npm install -g vercel
vercel
```

### Deploy to Netlify
```bash
npm run build
netlify deploy --prod --dir=.next
```
